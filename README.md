# Productive cherry trees
This mod adds special cherry tree variants that occasionally drop petals which when mined, drop valuable resources.

![Lots of trees ](https://cdn.modrinth.com/data/cached_images/dcae70cea034e0b3e4a3c935d020f235b31e8aef.png)

## Crafting
To create those variants, there's the sapling infuser that combines the blocks placed around it with a sapling placed on top. There's also another way to get other types of petals. Petal fusion which happens when one type of petals is under a tree of a different type. When the tree tries dropping petals there then the petals will fuse together into a third type. The varient's sapling can be obtained by combining the petals in a sapling infuser. If the full release version is ever made, most varients will be made using petal fusion and petal fusion recipe chains.

## Features
- A few tree variants (maybe later, many)
- Sapling infuser (and recipe type)
- Petal fusion (and recipe type)

## Trees
- Fire, made with lava and cherry sapling in sapling infuser
- Water, made with water and cherry sapling in sapling infuser
- Stone, made by Fire on Water petal fusion
- Gold, made rarely by Stone doing petal fusion with itself
- Test, creative variant (the pink one in screenshots)
